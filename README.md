# Anonymous Secrets
> Anonymous Secrets allows you to share your secrets anonymously.

![screenshot](https://alexmenlah.com/assets/images/projects/anonymousSecrets.png)

## Tools

- HTML/CSS
- Bootstrap
- Nodejs
- Express
- MongoDB
- Social Logins
- Authentication

## Usage

Install Dependencies

```
npm install
```

### Run

```
node app.js

Go to localhost:3000
```
